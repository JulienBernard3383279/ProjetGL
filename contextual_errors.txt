- Le type $NAME n'est pas defini. (0.2)

- La super class $NAME n'est pas une classe definie. (1.3)

- Le type d'un attribut de classe est void. (2.5)

- La methode possede une signature differente que la methode heritee. (2.7)
- La methode redefini ne retourne pas un sous type du retour de la methode heritee. (2.7)

- Le parametre est de type void. (2.9)

- La variable est de type void. (3.17)

- Appel a return dans une methode void. (3.24)

- Type incompatible de la rvalue. (3.28)

- Condition n'est pas de type  boolean. (3.29)

- Argument de print n'est pas de type int, float ou string. (3.31) 

- Types incompatibles pour une operation arithmetique. (3.49/50/52/53)

- Operandes de modulo n'est pas int. (3.51)

- Operande de l'operation booleene n'est pas de type boolean. (3.54/55)

- Operandes de la comparaison de types incompatibles (3.56/57/58/59/60/61)

- $NAME est cast en type incompatible. (3.39)

- Peut etre une erreur avec InstanceOf mais je ne sais pas (3.40)

- La class n'est pas definie. (3.42)

- Appel this en dehors d'une classe. (3.43)

- Operande de UnaryMinus n'est pas numerique. (3.62)

- Operande de Not n'est pas boolean. (3.63)

- Attribut 'protected' inaccessible. (3.66)
